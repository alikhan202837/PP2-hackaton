Detailed Description

@brief Class representing a button in a Pygame-based GUI.

This class creates a simple button with text that changes color on hover.

@param image pygame.Surface or None: The image to display on the button. If None, the button consists only of text.

@param textInput str: The text displayed on the button.
@param font pygame.font.Font: The font used to render the text.
@param baseColor tuple(int, int, int): The RGB color for the text when the button is not being hovered over.
@param hoveringColor tuple(int, int, int): The RGB color for the text when the button is being hovered over.
